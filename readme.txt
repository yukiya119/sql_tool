１ 起動すると、

    ===== Welcome to CRM Application =====
    [S]how: Show all users info
    [A]dd: Add new user
    [Q]uit: Quit The Application
    ======================================

    Your command >

までを表示。


２ 使えるコマンドは

    [S]how
    [A]dd
    [Q]uit

の３種類（それ以外は排他）


３ [S]を入力すると、ユーザーの一覧を表示
 - 追加したユーザーはどこに保存される？


４ [A]を入力すると、
    New user name >
    New user age >
    の順に入力を求め、age入力後に
    Add new user: ****
    をprint


５ 上記２以外のコマンドを入力すると
    *: command not found
    をprint

６ [Q]を入力すると
    Bye!
    をprint
